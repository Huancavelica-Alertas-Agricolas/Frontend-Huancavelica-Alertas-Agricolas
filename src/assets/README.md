# assets — Recursos estáticos

Qué incluir
- Iconos, logos, fuentes, imágenes optimizadas.

Convenciones
- Organizar por tipo: `icons/`, `images/`, `fonts/`.
- Evitar archivos muy grandes en el repositorio; usar CDN si es necesario.

Notas
- Importar assets desde `src` para que el bundler (Vite) los procese y aplique fingerprinting.
